# Flower Sales Dashboard

## 1. ภาพรวม (Overview)

แดชบอร์ดนี้ถูกสร้างขึ้นเพื่อวิเคราะห์และสรุปข้อมูลการขายดอกไม้ มีวัตถุประสงค์เพื่อให้เห็นภาพรวมของยอดขาย, ติดตามแนวโน้มการเติบโต, และเปรียบเทียบประสิทธิภาพการขายในมิติต่างๆ เช่น ช่วงเวลา, พื้นที่ (จังหวัด), และประเภทของสินค้า

## 2. องค์ประกอบหลักของแดชบอร์ด (Dashboard Components)

แดชบอร์ดประกอบด้วยส่วนแสดงผลหลัก 3 ส่วนคือ:

### 2.1. ภาพรวมหลัก (Key Metrics)

ตัวชี้วัดผลการดำเนินงานที่สำคัญ 4 อย่างจะแสดงอยู่ด้านบนสุดของแดชบอร์ด:
* **ยอดขายรวม (Total Sales):** มูลค่าการขายทั้งหมดในช่วงเวลาที่เลือก
* **จำนวนที่ขายได้ (Total Unit Sold):** จำนวนดอกไม้ทั้งหมดที่ขายได้
* **จำนวนออเดอร์ (Total Order):** จำนวนธุรกรรมการขายทั้งหมด
* **ยอดขายเฉลี่ยต่อออเดอร์ (Avg. Sales per Order):** ยอดขายเฉลี่ยที่เกิดขึ้นในแต่ละธุรกรรม

### 2.2. การแสดงผลด้วยกราฟ (Visualizations)

* **Year-over-Year Monthly Sales Comparison:** กราฟเส้นเปรียบเทียบแนวโน้มยอดขายรายเดือนระหว่างปีปัจจุบันและปีก่อนหน้า เพื่อให้เห็นการเติบโตหรือความเปลี่ยนแปลงในแต่ละเดือน
* **Flower Sales by Province:** กราฟแท่งแสดงยอดขายรวมในแต่ละจังหวัด เพื่อเปรียบเทียบว่าจังหวัดใดสร้างยอดขายได้สูงสุด
* **Flower Sales Distribution by Province:** กราฟวงกลมแสดงสัดส่วนเปอร์เซ็นต์ของยอดขายในแต่ละจังหวัดเทียบกับยอดขายทั้งหมด
* **Flower Purchases Ranked by Quantity:** ตารางข้อมูลที่แสดงรายละเอียดการสั่งซื้อ โดยเรียงลำดับจากปริมาณการซื้อสูงสุด เพื่อให้เห็นภาพรวมของออเดอร์สำคัญ

### 2.3. ตัวควบคุม (Controls)

* **ตัวกรองช่วงวันที่ (TransactionDate):** สำหรับเลือกช่วงเวลาของข้อมูลที่ต้องการวิเคราะห์
* **ตัวกรองภูมิภาค (Region):** สำหรับเจาะจงเพื่อดูข้อมูลเฉพาะภาคที่สนใจ

## 3. วิธีการใช้งาน (How to Use)

ผู้ใช้สามารถโต้ตอบกับแดชบอร์ดได้โดยการเลือกช่วงวันที่และภูมิภาคที่ต้องการจากเมนูตัวกรองด้านบนขวา กราฟและการคำนวณทั้งหมดบนแดชบอร์ดจะอัปเดตตามตัวกรองที่เลือกโดยอัตโนมัติ
